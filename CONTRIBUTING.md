# Contributing

This project is currently a personal learning project. Contributions may be accepted later after the core architecture becomes stable.

## Development flow

1. Create or pick an issue.
2. Create a branch from `main`.
3. Make a focused change.
4. Run `npm run typecheck` and `npm run build`.
5. Open a pull request.

## Commit message examples

```text
feat: add mock heart rate source
fix: keep overlay above normal windows
docs: explain Mi Band limitations
```
