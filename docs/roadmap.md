# 开发路线图

本文记录 HeartDock 的阶段目标、当前进度和后续计划。

HeartDock 当前处于 v1.0.0 前的正式候选收尾阶段。当前重点是验证 Windows 桌面心率悬浮窗、标准 BLE 心率读取、纯享显示模式、显示样式自定义、显示框样式预设、启动引导、自动更新、诊断导出和基础发布流程是否稳定，再进入 v1.0.0 稳定版收尾。

## 暂不优先处理的方向

以下内容暂时不作为近期重点：

- 小米手环私有协议适配
- 厂商私有认证密钥流程
- 跨平台 macOS / Linux 支持
- 云端同步
- 用户账号系统

这些方向不是完全放弃，而是当前阶段优先级低于标准 BLE、Windows 悬浮窗体验、自定义显示体验和安装包发布。
