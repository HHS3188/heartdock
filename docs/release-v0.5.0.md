# HeartDock v0.1.12 Release Notes

本版本是 HeartDock 的早期 Windows 预发布版本，重点完成显示样式、启动页、日夜模式、应用图标、安装包和发布体验整理。

---

## 主要更新

- 新增 HeartDock 应用图标
- 新增 Windows NSIS 安装包
- 新增启动页和首次启动说明
- 首次启动需等待 15 秒并确认阅读
- 后续启动显示启动页，用户手动点击进入主界面
- 新增免费开源说明、开发人员信息、源码链接和反馈入口
- 新增日间 / 黑夜模式切换
- 优化启动页和设置页窗口大小隔离
- 优化日间模式下的文字可读性、BLE 状态区域和心率显示区域
- 新增显示框样式预设
- 支持无背景、柔和玻璃卡片、圆角胶囊、霓虹直播框、二次元贴纸风
- 优化纯享模式命中范围
- 优化设置页控件、下拉菜单、输入框选中体验
- 优化右下角窗口缩放手柄体验

---

## 下载

普通用户请下载：

- `HeartDock Setup 0.1.12.exe`

安装后直接运行即可，不需要安装 Node.js 或 npm。

---

## 已知问题

- 鼠标穿透交互关闭时，在部分 Windows 环境下可能偶发退出。
- 如遇到退出，重新打开 HeartDock 即可。
- 建议暂时避免频繁快速切换鼠标穿透。
- 后续版本会继续排查 Electron 透明窗口和鼠标穿透状态切换的兼容性。
