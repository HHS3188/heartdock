# HeartDock v0.6.0 发布说明

## 概述

v0.6.0 是纯享模式和配置文件体验增强版本，重点改进全屏场景置顶、纯享拖动流畅度、显示框样式和设置迁移能力。

## 主要更新

### 纯享模式增强置顶（#58）

- 纯享模式与“始终置顶”同时开启时，会启用增强置顶。
- 增强置顶会定期恢复窗口层级，尽量保持在全屏 / 无边框全屏应用上方。
- 该能力仍受 Windows、独占全屏、游戏反作弊和受保护窗口策略限制。

### 纯享拖动体验优化（#59）

- 纯享模式拖动改为逐帧合并位移。
- 减少 mousemove 高频 IPC，提升拖动跟手度和流畅度。
- 保留原有双击退出纯享模式和鼠标禁止交互逻辑。

### 新增显示框样式（#59）

本版本新增 3 个显示框样式：

- 极光流光框
- 极简描边框
- 心电监护框

这些样式可在普通模式和纯享模式中使用。

### 配置导入导出（#59）

- 支持将当前配置导出为 JSON 文件。
- 支持从 JSON 文件载入配置。
- 载入时会校验并规范化配置字段，降低坏配置导致异常 UI 状态的风险。
- 载入配置时会自动关闭纯享模式和鼠标禁止交互，并断开 BLE 连接，避免误锁窗口或保留旧设备状态。
