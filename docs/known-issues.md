## 鼠标穿透关闭时偶发退出（v0.5.1 已修复/缓解）

### 现象

在部分 Windows 环境下，开启"鼠标穿透交互"后，再通过 `Ctrl + Shift + H` 关闭鼠标穿透时，HeartDock 可能偶发退出。

### v0.5.1 修复

v0.5.1 对该问题进行了稳定性修复：

- 主进程 `setOverlayClickThrough` 增加同值跳过，避免重复调用 `setIgnoreMouseEvents`
- 增加 200ms 最小切换间隔，防止快速 toggle 导致窗口不稳定
- 渲染进程纯享模式 hit-test 增加去重，减少重复 IPC 调用

修复后用户测试未再复现退出。如在 v0.5.1 仍遇到此问题，请提交 GitHub Issue 并注明 Windows 版本和复现步骤。

### 影响范围

- 主要影响鼠标穿透交互的频繁开关场景。
- 模拟心率、手动心率、启动页、显示样式、日间 / 黑夜模式、普通设置页不受直接影响。
