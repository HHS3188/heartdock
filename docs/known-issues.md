## 鼠标穿透关闭时偶发退出

### 现象

在部分 Windows 环境下，开启“鼠标穿透交互”后，再通过 `Ctrl + Shift + H` 关闭鼠标穿透时，HeartDock 可能偶发退出。

### 影响范围

- 主要影响鼠标穿透交互的频繁开关场景。
- 模拟心率、手动心率、启动页、显示样式、日间 / 黑夜模式、普通设置页不受直接影响。
- 如果应用退出，重新打开即可继续使用。

### 临时建议

- 不建议频繁快速切换鼠标穿透。
- 使用纯享模式时，优先通过心率本体拖动和双击退出。
- 如果误开启鼠标穿透，可以尝试按 `Ctrl + Shift + H` 关闭；如应用退出，重新打开即可。

### 后续修复方向

后续版本会继续排查：

- Electron 透明窗口与 `setIgnoreMouseEvents` 状态切换的兼容性
- 鼠标穿透与纯享模式命中区域之间的状态同步
- 主进程点击穿透切换保护和日志记录
- 在不同 Windows 版本、DPI、显卡环境下的复现条件
