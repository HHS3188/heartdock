# HeartDock v0.7.0 发布说明

## 概述

v0.7.0 是个性化场景、动态显示与心率记录版本，重点增强常用场景预设、显示框样式、心率动态效果和纯享模式记录导出。

## 主要更新

### 场景预设（#61）

- 新增 6 个常用场景预设：直播叠加、游戏极简、桌面陪伴、二次元贴纸、心电监护、办公低调。
- 预设会组合调整显示框样式、字号、颜色、发光和动态效果。
- 应用预设前使用应用内确认弹窗，避免误操作。
- 支持撤销上次预设，恢复应用预设前的显示配置。

### 新增显示框样式（#61）

本版本新增 3 个显示框样式：

- 像素游戏框
- 赛博扫描框
- 软萌云朵框

这些样式可在普通模式和纯享模式中使用。

### 动态显示（#61）

- 支持 BPM 更新时的心跳缩放。
- 支持心率区间颜色平滑过渡。
- 支持高心率提醒，可自定义提醒阈值。
- 支持低心率呼吸效果。
- 动效会遵守系统减少动态效果设置。

### 纯享心率记录（#61）

- 开启“记录纯享心率”后，进入纯享模式自动开始记录有效 BPM。
- 退出纯享模式时展示本次心率报告。
- 报告包含记录时长、平均心率、最高 / 最低心率、有效样本、心率波动、区间分布和时间线。
- 支持将本次报告导出为 PNG 图片。
- BLE 未连接或心率显示为 `--` 时不会写入无效样本。

### 配置与导出体验

- 配置导出默认文件名加入日期时间。
- 配置导入仍会安全关闭纯享模式、鼠标禁止交互和 BLE 连接，避免误锁窗口或保留旧设备状态。
